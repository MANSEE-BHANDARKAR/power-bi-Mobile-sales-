 # Mobile Sales Dashboard - Power BI Project

## Project Overview

This project is a Mobile Sales Dashboard developed using Power BI. It provides insights into mobile phone sales performance by analyzing sales, quantity sold, transactions, and trends across different brands, models, and locations. The dashboard helps users understand business performance through interactive visualizations.

## Objectives

* Analyze overall mobile sales performance.
* Track total sales, quantity sold, and transactions.
* Identify top-performing brands and mobile models.
* Monitor sales trends over time.
* Compare sales performance across different cities.
* Support data-driven business decisions.

## Tools and Technologies

* Power BI
* Power Query
* DAX
* Excel/CSV Dataset

## Dashboard Features

### Key Performance Indicators (KPIs)

* Total Sales
* Total Quantity Sold
* Total Transactions
* Average Sales Value

### Sales Analysis

* Brand-wise Sales Analysis
* Model-wise Sales Analysis
* City-wise Sales Performance
* Monthly Sales Trends

### Interactive Elements

* Filters and Slicers
* Dynamic Charts and Graphs
* Drill-down Analysis

## Skills Demonstrated

* Data Cleaning and Transformation
* Data Visualization
* Dashboard Development
* DAX Calculations
* Business Intelligence
* Data Analysis
* Reporting and Insights Generation

## Key Insights

* Identified the highest-performing mobile brands.
* Analyzed sales distribution across cities.
* Tracked monthly sales growth and performance trends.
* Evaluated transaction volumes and revenue generation.
* Compared product performance to identify business opportunities.

## Conclusion

This project demonstrates the use of Power BI to transform raw sales data into meaningful business insights. The dashboard enables efficient monitoring of sales performance and supports informed decision-making through interactive and visual reporting.
